# Contributing

Thank you for your interest in contributing to the netsuite-developer Codex skill.

## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Guidelines
- Maintain YAML frontmatter in SKILL.md
- Do not remove license headers
- Keep Python helper dependency-free
- Document new features clearly
